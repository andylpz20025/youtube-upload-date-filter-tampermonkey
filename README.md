# YouTube Upload Date & Age Filter (Tampermonkey)

Advanced search filter for YouTube:
- Sort by newest / oldest uploads
- Filter videos older than 10, 15 or 20 years
- Works on DE / UK / US YouTube

## Installation
1. Install Tampermonkey
2. Open the `.user.js` file in this repository
3. Click **Install**

## Features
- Buttons directly in YouTube search
- No external dependencies
- Pure client-side script

## Supported Languages
- German
- English (US / UK)
